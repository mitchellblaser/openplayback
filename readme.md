# OpenPlayback

Open Source Digital Signage software with fleet control

## Getting Started

1. **Install Dependencies**: `npm install`
2. **Start Developing**: `npm run dev`
2. **Build**: `npm run build && npm run dist`

## Contribute

1. **Clone project**: `git clone https://github.com/mitchellblaser/openplayback.git`
2. **Create feature/branch**: `git checkout -b feature/NAME`
