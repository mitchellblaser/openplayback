# OpenPlayback Client Dashboard

A simple React dashboard to manage and send commands to multiple OpenPlayback servers.

## Setup

1. Install dependencies:
   ```
   npm install
   ```
2. Start the development server:
   ```
   npm run dev
   ```

## Features
- Add multiple servers by name and URL
- Run commands (e.g., `/kill`, `/get-screenshot`) on any server
- View output or screenshots from server responses
